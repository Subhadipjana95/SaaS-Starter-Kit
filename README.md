# SaaS Starter Kit
A minimal Next.js starter template with shadcn/ui for building SaaS landing pages and dashboards.

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Motion
- shadcn/ui

## Installation

Node.js (18 or later) and npm are required.

#### Clone, install dependencies, and run in development:

```bash
git clone https://github.com/Subhadipjana95/SaaS-Starter-Kit.git
cd saas-starter-kit
npm install
npm run dev
```

#### Build and run production:

```bash
npm run build
npm start
```
